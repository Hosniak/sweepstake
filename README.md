# askporter Sweepstake Auto-Refresh Setup

This project is ready to publish as a free GitHub Pages site.

## How it works
- The page loads its latest content from sweepstake-data.json.
- A GitHub Actions workflow refreshes that feed and publishes the site.
- Anyone with the public GitHub Pages URL can view the latest version.

## What to do next
1. Push these files to a GitHub repository.
2. In the repository settings, open Pages and choose GitHub Actions as the source.
3. Make sure the default branch is main.
4. Run the Deploy sweepstake site workflow once.
5. Share the Pages URL that GitHub provides.
